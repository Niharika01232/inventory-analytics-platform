# Excel Business Intelligence Integration Guide
*Inventory Performance & Stock Optimization Analytics*

This document provides step-by-step instructions for business managers and supply chain users to implement an interactive analytics spreadsheet using Excel, importing the clean datasets generated by the Python pipeline.

---

## 1. Data Connection & Power Query Setup

To maintain a dynamic link between the Python processing engine and Excel:
1. Open Excel and create a new workbook named `Inventory_Operations_Dashboard.xlsx`.
2. Go to the **Data** tab → **Get Data** → **From File** → **From Text/CSV**.
3. Select `data/processed/sales.csv` and click **Import**.
4. In the Power Query preview dialog, click **Transform Data**.
5. Ensure data types are correctly detected:
   * `sale_date`: Date
   * `quantity_sold`: Whole Number
   * `selling_price_per_unit`: Decimal Number
   * `revenue`: Decimal Number
6. Click **Close & Load To...** → Select **Only Create Connection** and check **Add this data to the Data Model**.
7. Repeat the same steps for:
   * `products.csv`
   * `inventory_recommendations.csv`
   * `suppliers.csv`

## 2. Advanced Lookup Functions (XLOOKUP & INDEX MATCH)

To build a **Product Search Portal** where team members can query SKU health instantly:

### Method A: Using XLOOKUP (Recommended for Excel 365/2021)
Create an interface where a user inputs a `product_id` in cell `B3` and Excel retrieves:
* **Product Name** (Cell `C3`):
  ```excel
  =XLOOKUP(B3, products.csv[product_id], products.csv[product_name], "Product Not Found")
  ```
* **Category** (Cell `D3`):
  ```excel
  =XLOOKUP(B3, products.csv[product_id], products.csv[category], "N/A")
  ```
* **Stock Status** (Cell `E3`):
  ```excel
  =XLOOKUP(B3, inventory_recommendations.csv[product_id], inventory_recommendations.csv[health_label], "No Status")
  ```

### Method B: Using INDEX & MATCH (For backward compatibility)
To search for the **Supplier Name** associated with the `product_id` in `B3`:
```excel
=INDEX(suppliers.csv[supplier_name], MATCH(INDEX(products.csv[supplier_id], MATCH(B3, products.csv[product_id], 0)), suppliers.csv[supplier_id], 0))
```
*Explanation*:
1. The inner `MATCH(B3, products.csv[product_id], 0)` locates the row of the product.
2. The inner `INDEX` retrieves the `supplier_id` from that row.
3. The outer `MATCH` locates that `supplier_id` in the `suppliers.csv` table.
4. The outer `INDEX` returns the corresponding company name.

## 3. Data Validation for Procurement Orders

To prevent data entry errors in manual procurement logs:
1. Select the cells in your **New Orders Log** sheet under the column `Warehouse ID`.
2. Navigate to **Data** tab → **Data Tools** group → **Data Validation**.
3. Under **Allow**, select **List**.
4. In the **Source** field, select the range containing the warehouse IDs:
   `=warehouses.csv!$A$2:$A$6` (or Chicago, Atlanta, Newark, Los Angeles, Dallas).
5. Click the **Error Alert** tab:
   * Title: `Invalid Warehouse Code`
   * Error Message: `Please select a valid warehouse code from the drop-down list. Unauthorized locations are blocked.`

## 4. Building Pivot Tables & Slicers

### Pivot Table 1: Category Revenue & Profit Analysis
1. Select any cell inside the imported `sales` table.
2. Go to **Insert** → **PivotTable**. Select **New Worksheet** and name it `Analytics_Pivot`.
3. In the PivotTable Fields pane:
   * Drag `category` (from Products) into **Rows**.
   * Drag `revenue` (from Sales) into **Values** (summarized as **Sum**).
   * Drag `revenue` again into **Values** → Right-click → **Show Values As** → **% of Grand Total**.
4. Format columns as Currency (`$#,##0`).

### Pivot Table 2: Warehouse Stock Valuation
1. Create a Pivot Table using the `inventory_recommendations` connection.
2. Drag `warehouse_name` into **Rows**.
3. Drag `stock_on_hand` into **Values** (Sum).
4. Drag `health_label` into **Columns**.

### Connecting Slicers
1. Click on Pivot Table 1.
2. Go to **PivotTable Analyze** → **Insert Slicer**.
3. Select `category`, `warehouse_name`, and `channel`.
4. Right-click the newly created Slicer → **Report Connections...**.
5. Check the box for both Pivot Table 1 and Pivot Table 2. Now, slicing by "Online" or "North Hub" will update all visuals simultaneously!

## 5. Conditional Formatting Rules

Apply these visual cues on the `Inventory Status` table:
1. Select the range under the `health_score` column.
2. Go to **Home** → **Conditional Formatting** → **Manage Rules** → **New Rule**.
3. Choose **Format all cells based on their values** and select **3-Color Scale**:
   * Minimum: Value `0` (Color: Soft Red `#FFC7CE`)
   * Midpoint: Value `70` (Color: Soft Yellow `#FFEB9C`)
   * Maximum: Value `100` (Color: Soft Green `#C6EFCE`)
4. Select the range under the `health_label` column.
5. Create a new rule: **Format only cells that contain**:
   * Cell Value equal to `Critical` → Apply Bold Red Text, Light Pink Fill.
   * Cell Value equal to `Excellent` → Apply Bold Dark Green Text, Light Green Fill.

## 6. Dynamic Chart Design

Create a **Category Sales Split** chart that adjusts based on slicer selections:
1. Select Pivot Table 1.
2. Go to **Insert** → **PivotChart** → Select **Clustered Bar Chart**.
3. Clean the chart layout:
   * Right-click the field buttons on the chart and choose **Hide All Field Buttons on Chart**.
   * Double-click the gridlines and delete them to give a flat, premium design.
   * Apply a sleek blue color palette to the bars matching the corporate theme.
   * Add a dynamic chart title linked to cell `$A$1` (which calculates the current filter value).
